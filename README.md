# 📊 Análise Exploratória de Dados – Saúde Mental

## 📌 Sobre o Projeto

Este projeto consiste em uma análise exploratória de dados aplicada a um conjunto fictício de informações sobre saúde mental. O objetivo foi compreender padrões demográficos e clínicos da amostra, estruturando a análise de forma organizada e profissional por meio de uma aplicação interativa desenvolvida em Streamlit.

O projeto foi construído com foco em:

* Estrutura analítica clara
* Organização por etapas
* Visualizações coerentes
* Síntese interpretativa dos resultados
* Encerramento formal com conclusões

---

## 🎯 Objetivos da Análise

* Explorar o perfil demográfico dos pacientes
* Identificar os diagnósticos mais frequentes
* Analisar níveis médios de estresse
* Avaliar uso de medicação
* Verificar padrões de faltas e consultas
* Cruzar diagnóstico por gênero
* Avaliar satisfação com o tratamento

---

## 📂 Estrutura da Aplicação

A aplicação foi organizada em abas temáticas:

### 1️⃣ Visão Geral

* Total de pacientes
* Idade média
* Distribuição por gênero
* Estado civil
* Escolaridade

### 2️⃣ Perfil Clínico

* Diagnóstico principal
* Uso de medicação
* Número médio de consultas
* Faltas
* Nível médio de estresse
* Estresse por diagnóstico
* Distribuição proporcional de diagnóstico por gênero
* Satisfação com o tratamento

### 3️⃣ Conclusões

* Síntese acadêmica da análise
* Interpretação dos principais achados
* Identificação do autor
* Informações de versão e licença

---

## 🔎 Principais Resultados

* Amostra composta por 2.000 pacientes
* Idade média aproximada de 44 anos
* Distribuição equilibrada entre masculino e feminino
* Grupo não binário com menor representatividade
* TEPT como diagnóstico mais frequente
* Transtorno do Pânico como menos frequente
* 55,65% dos pacientes fazem uso de medicação
* Nível médio de estresse: 5/10
* Maioria dos pacientes apresenta apenas uma falta anual
* Ansiedade apresenta maior nível médio de estresse
* Grupo não binário demonstrou menor nível médio de satisfação com tratamento


3. Crie e ative um ambiente virtual (opcional, mas recomendado)

4. Instale as dependências:

```
pip install -r requirements.txt
```

5. Execute a aplicação:

